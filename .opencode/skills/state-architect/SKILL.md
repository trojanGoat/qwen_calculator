# Skill: State Architect & Headless Logic

## Trigger
Use this skill whenever you are asked to build a new feature, functional dashboard, or complex component.

## Workflow
1. Act as a backend/state engineer. 
2. Write the custom React hooks, the mock data API functions, and the required state management (e.g., Zustand store).
3. Map out the component tree.
4. **CRITICAL:** Do not write any UI components yet. Wait for the user to approve the architecture before proceeding to the UI phase.

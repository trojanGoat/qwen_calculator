# Skill: UI Execution

## Trigger
Use this skill only AFTER the State Architect logic has been approved, or when the user explicitly asks for UI component generation.

## Workflow
1. Act as the UI engineer.
2. Build the UI components and wire them up to the custom hooks and state management that were just written.
3. Strictly enforce the global `AGENTS.md` rules: Use Tailwind CSS utility classes, Lucide-React icons, and emulate the Shadcn aesthetic (clean, high whitespace, Slate/Zinc palette).

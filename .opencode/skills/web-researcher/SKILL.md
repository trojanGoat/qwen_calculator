# Skill: Web Researcher

## Trigger
Use this skill when you encounter an error you do not understand, need the latest syntax for a framework, or when the user explicitly asks you to search the web or check online for solutions.

## Workflow
1. You have autonomous terminal access. Use bash tools like `curl` or `wget` to fetch documentation from known, reliable sources. 
2. If you need to search for a specific error, write and execute a quick Python script to scrape relevant StackOverflow or documentation pages.
3. Read the terminal output and summarize the findings before attempting to write code based on the new information.
